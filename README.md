# Unit Price Calculator

Category: Decision Making

A lightweight single-page decision-making tool for comparing unit prices across purchase channels. It helps record product prices, base amount, bonus amount, total cost, inventory, usage pace, expiry, historical lows, and cart selections so the best buying option is easier to identify.

## Versions

- `v3-precision.html` - Chinese precision version.
- `v3-precision-en.html` - English precision version.
- `v3-precision-bilingual.html` - Chinese / English switchable version.
- `index.html` - Version entry page.
- `v1-editorial.html` and `v2-apothecary.html` - Earlier visual variants.

## Features

- Add multiple purchase channels for one product.
- Calculate unit price from total price, base amount, and bonus amount.
- Rank current offers together with historical low records.
- Keep product units consistent across the same product.
- Track inventory, usage period, estimated runway, and expiry.
- Select items into a cart-style comparison view.
- Export current product price records to CSV.
- Responsive layout for desktop and small screens.

## How to Use

Open `v3-precision-bilingual.html` in a browser for the latest switchable version, or use `v3-precision-en.html` for an English-only interface.

All data is stored locally in the browser with `localStorage`; no server is required.
